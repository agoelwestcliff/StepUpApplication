# StepUpApplication
A networking app for like-minded professionals

## Development

This project uses React with Vite. Follow these steps to run the app on your
computer.

### Prerequisites

Install Node.js before running the project. Node.js includes `npm`, which is used
to install and run the app.

You can confirm Node.js and npm are installed with:

```bash
node -v
npm -v
```

### Run the app

From the project folder, install the project dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

After the server starts, Vite will print a local URL in the terminal. Open that
URL in your browser. It is usually:

```text
http://127.0.0.1:5173
```

### Stop the app

In the terminal where the app is running, press:

```text
Control + C
```

### Build the app

To create a production build, run:

```bash
npm run build
```

The generated files will be created in the `dist` folder.
